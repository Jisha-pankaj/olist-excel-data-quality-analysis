# olist-excel-data-quality-analysis
An Excel project analyzing data quality in the Olist dataset, covering raw data assessment, data overview, and comprehensive documentation of data issues with validation checks.

## Dataset
- Source: Olist dataset (orders-related files)
- Tool used: Microsoft Excel

## Project Objective
- Understand raw business data
- Identify data quality issues such as missing values, duplicates, invalid formats, and logical inconsistencies
- Document issues clearly for analysis and reporting
